
@file
Suggested Tags.

Extends the tag input type with the option of extracting values from a JSON endpoint.

If $vars['endpoint'] is given to the tag, then the endpoint will be queried with a variable 'term'
containing the tag so far typed. The endpoint should return a JSON array of matching tags.
